== Description ==
Elementor Advance Widgets is a powerful extension that adds advanced custom widgets to Elementor Page Builder, enabling you to create more dynamic and feature-rich pages.

== Features ==
* Custom advanced widgets for Elementor
* Easy-to-use interface
* Fully responsive design
* Regular updates and improvements

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/elementor-advance-widgets/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Elementor editor and find the new widgets in the widgets panel

== Frequently Asked Questions ==

= Does this plugin require Elementor? =
Yes, you must have Elementor installed and activated.

= Is it compatible with Elementor Free version? =
Yes, it works with both Elementor Free and Pro versions.
